Capstone Project - Student Management System

This Web- based application is designed to simplify academic management processes. It support two user roles:

  • Admin - Can manage students, departments, courses, enrollment and payments.
  • Student - Can manage their profile, enroll in courses, and view payment history.

Key Features:

  1. Secure login for both Admin and student roles.
  2. Dashboard with real-time counts (Students, enrollments, courses statuses)
  3. CRUD operations for courses, departments and students.
  4. Enrollment and Payment tracking
  5. Password Change functionality
  6. Sorting and filtering for quick data access

Technologies Used:

  • React, Node.Js, Express, MySQL, Bootstrap
